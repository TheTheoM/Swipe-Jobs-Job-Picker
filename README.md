# Swipe Jobs Job Picker

## Installation
### Clone Repo
```bash
git clone https://github.com/TheTheoM/Swipe-Jobs-Job-Picker.git
cd Swipe-Jobs-Job-Picker
```

### Copy .env.example to .env
Enter credentials from the email.
```bash
VITE_WORKER_ID=''
VITE_API_BASE_URL=''
VITE_LOCATION_IQ_ACCESS_TOKEN=''
```
### Run Frontend
```bash
npm install
npm run dev
```
Visit typically http://localhost:5173 in your browser


## Run Tests

```bash
npm test
```
## Run Project Health Check
Runs a full scan for issues across dependencies, TypeScript, and linting.
```bash
npm run check
```
